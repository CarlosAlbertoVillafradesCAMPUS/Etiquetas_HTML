Etiquetas Html

Etiquetas de imagen y multimedia

<'audio'>---El elemento audio se usa para insertar contenido de audio en un documento

<'img'>--- representa una imagen en el documento

<'track'>---se utiliza como elemento hijo de los elementos multimedia, audio y video. Le permite especificar pistas de texto cronometradas (o datos basados en el tiempo), por ejemplo, para manejar subtítulos automáticamente.

<'video'>---se utiliza para incrustar vídeos en un documento HTML o XHTML.

Etiquetas Tablas

<'table'>---representa datos en dos o mas dimensiones.

<'td'>--- define la celda de una tabla que contiene datos. Esta participa en el modelo de tablas

<'th'>---define una celda como encabezado de un grupo de celdas en una tabla.

<'tr'>---define una fila de celdas en una tabla. Estas pueden ser una mezcla de elementos td y th.

<'col'>---Permite especificar propiedades para una columna o un grupo de ellas.
